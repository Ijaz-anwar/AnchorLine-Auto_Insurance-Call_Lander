# AnchorLine Insurance

This project is a modern auto insurance landing page built with React, TypeScript, and Tailwind CSS.

## Features

- **Responsive Design**: Works seamlessly on desktop and mobile.
- **Interactive Chat Widget**: Engages users to provide insurance quotes.
- **Legal Pages**: Includes Privacy Policy and Terms of Service.
- **Optimized Performance**: Built with Vite for fast load times.

## Getting Started

1.  Clone the repository.
2.  Run `npm install` to install dependencies.
3.  Run `npm run dev` to start the development server.
